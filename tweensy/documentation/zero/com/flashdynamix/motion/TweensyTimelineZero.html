Packagecom.flashdynamix.motion
Classpublic class TweensyTimelineZero

TweensyTimelineZero is used by the TweensyZero class as a simple version of the TweensyTimeline. The TweensyTimelineZero Class has events for:
  • onComplete - When all tweens in the TweensyTimelineZero Class are done
  • onUpdate - When all animations in the TweensyTimelineZero Class have been updated.
These events allow for predefined params to be applied when they are executed. Tweensy favours this method as it allows for predefining the event and the params it requires without all the fussiness of Event listeners.
